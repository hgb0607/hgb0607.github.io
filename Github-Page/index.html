关于GitHub

GitHub是通过Git进行版本控制的软件源代码托管服务，是一个基于git 的社会化代码分享社区，所谓social coding。
通过GitHub，你可以

  • 在Github上创建免费的远程仓库(remote repository)，分享你的代码，当然也可以关注其他人的代码
  • 建立公司账户，创建私有的远程仓库，与开发团队共同协作开发
  • 使用Github Pages托管个人静态网站
    等等

使用Github Pages托管静态网页

(这是一个简单的托管静态网页的教程，只是把你写好的html网页托管到GitHub上，不教授html怎么写。而且这只是个简易教程，也可以使用git命令或者github桌面程序来实现，但在此并不讨论。)

注册一个GitHub账号

首先，需要在GitHub 上注册一个属于自己的账号。过程比较简单，输入github.com ，输入自己的用户名、邮箱、密码，然后点击“Sign up for GitHub”按照提示进行注册即可（需要验证邮箱）。
（一般情况下，只需要注册免费账号就已经足够使用了）

创建仓库

  1. 点击右上角的 + , 在弹出的下拉菜单里, 点击 new repository .

  2. 填写相应的信息，创建你的GitHub pages项目仓库

    注意：图中使用的是name.github.io,是指注册时用户名为name的情况。假如你注册是用户名为”zhangsan”，则为”zhangsan.github.io”。
